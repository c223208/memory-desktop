## 概要
本プロジェクトは、ブラウザ上でデスクトップ環境を模した操作体験を提供するwebアプリケーションである。
複数ウィンドウの生成・移動・管理を可能とし、視覚的再現ではなく操作感と構造設計を重視している。

## 背景
本システムは、ゼミでお世話になった教授へ感謝の気持ちを伝えることを目的として制作を開始した。
単に文書を表示するのではなく、あえて操作の手間を介在させ、探索・発見する楽しさを生むことを意図した。
